# Student Management System

## Description

This is a Student Management System developed in C. The program helps manage student records through a menu-driven interface.

## Features

* Add student details
* View student records
* Search for a student
* Update student information
* Simple and user-friendly menu system

## Technologies Used

* C Programming Language
* GCC Compiler
* Visual Studio Code

## How to Compile

```bash
gcc student_management_details.c -o student_management.exe
```

## How to Run

```bash
.\student_management.exe
```
